# TECHHUB

**TECHHUB** — это веб-сайт интернет-магазина компьютерных девайсов (наушники, мышки, клавиатуры, коврики), разработанный для учебного проекта. Сайт реализован с современным дизайном и удобным интерфейсом, позволяющим пользователям просматривать каталог товаров, оформлять заказы и отслеживать статистику продаж.

---

## Стек технологий

- **Фронтенд:** React, Tailwind CSS, Ant Design, Vite  
- **Бэкенд:** Node.js, Express  
- **База данных:** SQLite3  
- **Контроль версий:** GitVerse  

---

## Функциональные возможности

- Каталог товаров с разделением по категориям  
- Корзина и оформление заказов  
- Регистрация и авторизация пользователей  
- Просмотр статистики продаж товаров  
- Адаптивный интерфейс для всех типов устройств
